# Equity Research & Valuation Report: Marico Ltd. (NSE: MARICO)

## 📊 Project Overview
This repository hosts a fully integrated, audited 3-statement financial model and an Unlevered Free Cash Flow (UFCF) Discounted Cash Flow (DCF) model for Marico Ltd. The explicit forecast period spans **FY2027 to FY2031**.

## 🧠 Strategic Investment Thesis
* **The Premiumization Trap:** While Marico dominates the branded coconut oil space (~63% volume share), standard 5-year explicit projection windows heavily penalize its capital-light growth.
* **Valuation Disconnect:** My base-case model derives an intrinsic value of **₹348 per share**, signaling that the current market price (~₹645) factors in highly aggressive long-term assumptions.

## ⚙️ Valuation Methodology & Infrastructure
* **Discount Rate (WACC):** Dynamically calculated using localized country risk premiums and an adjusted emerging-market Equity Risk Premium (ERP).
* **Sensitivity Framework:** Stress-tested using a two-variable data table evaluating changing combinations of WACC vs. Perpetual Growth Rate (g).
